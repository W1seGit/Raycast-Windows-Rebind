Raycast Windows Rebind

Remap the Windows key to open Raycast instantly, without losing your shortcuts.

Download .exe View on GitHub

How it works

  • Press Win alone to open Raycast (triggers Alt+Space).
  • Press Win + E, Win + Tab, etc. for normal Windows shortcuts.
  • Zero config required. Runs silently in the tray.

Installation

  1. Download RaycastRebind.exe.
  2. Run it. (A tray icon will appear).
  3. To run on startup: Press Win+R, type shell:startup, and paste a shortcut to the exe there.

Customization

The default hotkey sent is Alt+Space. To change it:

  1. Edit main.ahk.
  2. Change LAUNCHER_HOTKEY := "!{Space}".
  3. Recompile or run with AutoHotkey v2.
